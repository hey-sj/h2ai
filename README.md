To Run: 
1. Clone the Repo
2. In terminal, cd into the backend folder.
3. run source venv/bin/activate # Mac Linux   or  venv/Scripts/activate # Windows
4. run python app.py
5. cd into the frontend directory
6. run npm run dev
7. navigate to http://localhost:5000 in your browser
